# Hiring Talent 

Build a scalable hiring process — from knowing when to hire to sourcing candidates and running interviews that reveal top talent.

At the end of this module, you will have built the foundation of a scalable hiring process. You will know when to hire, how to define roles, source and assess candidates, and how to run interviews and reference calls that consistently identify exceptional talent.

## Common Mistakes

- Hiring the wrong roles at the wrong time. 
- Confusing great résumés with great fits.
- Running unstructured or biased interviews.

**Duration:**  ~4 weeks
