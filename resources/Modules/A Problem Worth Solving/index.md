# A Problem Worth Solving

Validate whether the problem you identified is worth solving through need-finding, market sizing and competitor analysis.

At the end of this module, you will have identified a problem or opportunity worth pursuing, that you are excited about, summarised in a problem statement with which you can start with in Ideating Solutions.

## Common Mistakes

- Not solving a market need
- Lack of passion for the opportunity
- Creating a solution in search of a problem
